# Prasannara-Student Management System – Theory
This program is a basic Python application designed to manage student information. It allows the user to perform simple operations such as adding student details, viewing all stored records, and searching for a specific student.

The program uses a list as the main data structure to store student records. Each student’s information is stored in the form of a dictionary, which contains keys like name, age, and marks. This makes it easy to organize and access data.

The program is divided into different functions, where each function performs a specific task. One function is used to add new student data by taking input from the user. Another function is used to display all student records in a readable format. A separate function is used to search for a student by name, making it easy to find specific information.

A menu-driven approach is used in the program. This means the user is shown a list of options and can choose what action to perform. Based on the user’s choice, the program calls the appropriate function. The menu continues to run in a loop until the user chooses to exit the program.

The program also uses loops and conditional statements to control the flow of execution. Loops help in repeating actions like displaying the menu or going through student records, while conditional statements help in making decisions based on user input.
